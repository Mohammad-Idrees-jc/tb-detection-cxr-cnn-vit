# TB Detection from Chest X-rays

This project focuses on detecting Tuberculosis (TB) using deep learning techniques.

Current stage:
- CNN baseline model implementation
- Enhanced model with CLAHE preprocessing and DenseNet121 (CheXNet)
- PyTorch-based implementation with pretrained DenseNet121
- Grad-CAM visualization for model interpretability

Future work:
- Improve CNN architecture
- Implement CNN + Vision Transformer hybrid model
